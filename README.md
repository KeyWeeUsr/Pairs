# Pairs
Pairs is a simplified memory game in python with Kivy. The goal was to make it simple for a beginner, yet still fully functional.

#### How to use
###### Requirements
- kivy

###### Windows/Linux
- Install kivy
- Run `main.py`

###### Android
- Use KivyLauncher 
- Use buildozer to package APK and install directly
